# 🚀 InsightIQ

> AI-Powered Business Intelligence & Data Analytics Platform

InsightIQ is an intelligent analytics platform that enables users to upload datasets or PDF reports and instantly receive AI-powered insights, interactive dashboards, business recommendations, and downloadable reports.

---

## ✨ Features

- 📊 CSV Dataset Analysis
- 📈 Excel Dataset Analysis
- 📄 PDF AI Report Analysis
- 🤖 AI Business Analyst
- 📉 Interactive Charts & Dashboards
- 📑 Executive Reports
- 📥 PDF & Excel Export
- 📱 Responsive Design
- 🌙 Modern UI

## ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/jawwadjtq/Insight-IQ.git
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```
